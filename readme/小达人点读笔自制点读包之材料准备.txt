前面说过 小达人点读笔使用入门，后续就是进阶使用。

进阶，就是自制点读包，我自己常用的有两种方式。

一、通过录音贴，制作点读书籍，好处是直接点贴纸，播放音频。

二、通过书名贴+内容贴，制作点读书籍。好处是音频贴纸相对来说容易管理，简单说，就是一本书（一个书名贴）对应一堆内容贴。

制作点读包之前，我们要先做好准备工作，磨刀不误砍柴工！具体来说，就是音频、软件、贴纸三件套。



一、音频

音频可以使用别人已录制好的音频，当然假如你对其他人出品的音频质量不满意，抑或你是个声音主播，也可以自己录制。

很多畅销的百科和绘本，出版社官方可能出了音频，甚至视频，比如《那些重要的事》 DK 出了视频，《不一样的卡梅拉》则由二十一世纪出版社录制了音频。这类音视频质量非常高！有趣又好听。所以，建议在买书之前，了解下是否有对应的音频，试听一下，如果音频质量很高，大概率是一本畅销书，也大概率是本好书。
有些书，在出版社未提供音视频，去喜马拉雅、网易云、B 站类似的音视频平台去找找。如果一本受众比较广的书，大概率是有主播或第三方录制的音视频。比如《小兔汤姆》系列绘本。
在以上都没有的时候，你就可以考虑为自己的孩子录制一段。
选书的标准还是要书本身质量过关，参考大家的推荐去买。不要在意是否有音视频。有，也只是锦上添花！



二、软件

说到软件，分为三类：

一类是音频处理的软件，一类是麦芽官方出的点读包制作和安装客户端，最后是重命名软件。

先来说说音频处理。主要用于：将视频或其他格式的音频转换为 MP3、音频切割剪辑成单个片段用于与贴纸对应。
格式转换软件：格式工厂（Win）、HandBrake（Mac）

音频剪辑：GoldWave、Audacity

因为 Nemo 爸爸是个程序员，用的 MacBook，所以这个音频转换和切割软件是 ffmpeg。

2. 麦芽官方的点读包制作和安装软件，去 http://maiya.com 下载即可。必须要吐槽的是，官方出品的客户端软件 UI 界面古老而且没有适配好，在很多电脑上界面错乱严重。

3. 重命名软件

为什么要重命名，点读包文件名需要固定的格式，比如 0001 这种序号。各个操作系统本身都支持简单的重命名，但如果遇到文件名乱七八糟，重命名软件还是很有必要的。

mac 系统 Nemo 爸爸用的是 Renamer。



三、贴纸

贴纸在上一篇小达人点读笔使用入门已经说过了。如果贴纸不够，去 pdd 买吧。

以上工具和材料准备好了，就只剩实际操作了！
