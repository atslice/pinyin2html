小达人自制点读包的方法
https://zhuanlan.zhihu.com/p/127343981

小达人点读笔相对于其他品牌点读笔的最大优势就是，它是一个开放的资源平台，平台上有很多小达人爱好者或者妈妈们自制的点读包可以供大家使用。除了购买小达人系统点读书，还有很多图书支持小达人点读，买书自带点读包，安装即可。如果没有现成的点读包，也可以自制点读包，把一本普通图书变身小达人点读笔支持的有声读物，这里以儿童绘本《我妈妈》为例介绍如何自制点读包。

1. 找音频

对应图书的音频来源一是在网上搜索，比如喜马拉雅app中有很多图书的有声资源，另外就是自己录音，小达人点读笔有录音功能，可以自制音频。

2. 剪切音频

一般情况下，找到的音频都是一个完整的音频，需要我们剪切出和书中每页文字相匹配的一段一段的音频，剪切MP3格式的音频可以下载一个MP3ABCut音频剪切软件，这个软件小巧易操作，没有音质损失。

1.打开MP3ABCut软件

2.点击“打开”按钮，导入音频

3.按定位键“↑”设好a点、b点后（a点为剪切音频的起始，b点为剪切音频的末尾），即开始在ab间复读，可以试听剪切效果

4.剪贴段落选取无误后，按“tab”键进行剪切，选择保存路径和命名，一般命名就是按数字顺序，比如001、002…

5.再次按“↑”键取消a点、b点，用同样的方法剪切下一段音频

音频剪切完后可以保存在桌面的一个文件夹中，方便使用。

3. 合成

1.小达人点读笔关机状态下，用数据线连接电脑，打开小达人点读笔客户端

2.在客户端菜单栏选择第四项“附件功能”，然后选择“智能自制内容工具”

3.在智能自制内容工具对话框，选择“新建项目”，名称：输入与图书相同的名字，位置：选择合成文件保存的位置，点“确定”

4.选择“批量添加”，把剪切好的音频导入

5.在列表中把书名对应的音频找到，选择“删除音源”（因为书名音频要单独设定）

6.在封面声音：选择“未设定”，导入书名的音频文件

7.生成小达人点读笔文件类型选择“普通自制教材”，点击“生成”

8.此时保存的路径下面就会生成一个.dab格式的点读包，双击点读包，选择可用的书名贴号，以及可用的内容贴起始号，点击“安装”

9.安装完成，安全退出小达人点读笔

10.用小达人点读笔点击书名贴和内容贴，试听效果，无误后贴到图书的对应页面上即可。
